Data Binding

Data binding is a mechanism to coordinate the communication between a component and its template.

There are four types of data binding in Angular:

  • Interpolation
  • Property Binding
  • Event Binding
  • Two-way Binding

Interpolation

Interpolation is a one-way data binding technique that allows you to display component data in the template. It uses double curly braces to bind data from the component to the template.

FirstName: {{ firstName }}
LastName: {{ lastName }}

isDisables: {{ isDisabled }}

Property Binding

Property binding is a one-way data binding technique that allows you to bind a property of a DOM element to a component property. It uses square brackets to bind data from the component to the template.



Event Binding

Event binding is a one-way data binding technique that allows you to bind an event of a DOM element to a component method. It uses parentheses to bind data from the template to the component.

Two-way Binding

Two-way binding is a combination of property binding and event binding. It allows you to bind data from the component to the template and vice versa. It uses square brackets and parentheses to bind data from the component to the template and vice versa.
