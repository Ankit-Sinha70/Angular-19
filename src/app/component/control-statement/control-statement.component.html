Control Statement

Control statements are used to control the flow of execution in a program. In Angular, we can use structural directives like *ngIf, *ngFor, and *ngSwitch to implement control statements.

In this example, we will use *ngIf to conditionally display a message based on the value of a variable.

In this example, we will use *ngFor to iterate over an array of items and display them in a list.

In this example, we will use *ngSwitch to display different messages based on the value of a variable.
