Attribute Directives

Attribute directives are used to change the appearance or behavior of an element, component, or another directive. They are typically used to modify the properties of an element or to add event listeners.

ngClass

The ngClass directive is an Angular built-in directive that allows you to conditionally apply CSS classes to an HTML element based on the evaluation of an expression.
