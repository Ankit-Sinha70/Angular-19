What is the use of nfFor

ngFor is a structural directive in Angular that allows you to iterate over a collection (like an array) and render a template for each item in the collection.

It is commonly used to display lists of data in a dynamic and efficient way.

ngFor is a powerful directive that can be used to create dynamic lists, tables, and other repeating structures in your Angular applications.

It is a part of Angular's template syntax and is used to create dynamic views based on data.

In this example, we will use ngFor to display a list of cities.

  • {{ city.name }} - {{ city.population }}
